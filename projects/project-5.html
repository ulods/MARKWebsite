All In One Editor

Our project page

Project Description

All In One Editor is a system useful to design relational databases. All In One Editor is for novice and advanced users: it allows editing the schema directly in the diagram, explore the data per mouse, visually edit queries, design schema offline, deploy and upgrade the schema on different servers, build database reports and more..
